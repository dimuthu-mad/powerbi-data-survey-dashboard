# Data Professional Survey Dashboard - Power BI

Interactive Power BI dashboard analyzing survey data from data professionals.

## Features
- Salary analysis by job title
- Country-wise survey breakdown
- Favorite programming languages
- Career satisfaction insights
- Interactive filters and slicers

## Tools Used
- Power BI
- Power Query
- DAX (Basic)
- Data Visualization
